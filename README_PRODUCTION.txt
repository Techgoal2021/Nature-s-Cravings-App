# Nature's Cravings MVP - Launch Guide

This project has been hardened and optimized for production deployment as "Nature's Cravings". 

## Step 1: Clone for Production
Since this project started as "Petaverse Assignment-5", we recommend creating a dedicated folder for your MVP:
1. Open your Mac's **Finder**.
2. Navigate to: `Desktop/Petaverse Cohort2 Fullstack course/`.
3. Right-click on the `Petaverse Assignment-5` folder and select **Duplicate**.
4. Rename the duplicate copy to **"Nature's Cravings"**.

## Step 2: Deployment
We have included a `vercel.json` file for an effortless launch.
1. Sign up/Login to [Vercel](https://vercel.com).
2. Click "Add New" > "Project".
3. Upload/Select your new "Nature's Cravings" folder.
4. Add your **GEMINI_API_KEY** and any other sensitive keys to the "Environment Variables" section in the Vercel Dashboard.
5. Click **Deploy**.

## Step 3: Enjoy!
Your premium organic dining platform is now live!
