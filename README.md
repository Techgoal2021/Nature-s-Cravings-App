# 🌿 Nature's Cravings - Pure Organic Dining MVP

**Nature's Cravings** is a high-end, full-stack web platform designed to bring the luxury of organic dining to the digital space. Built with a focus on premium aesthetics and seamless user interaction, the platform provides a complete end-to-end experience for health-conscious diners.

## ✨ Key Features
*   **🤖 Smart AI Concierge**: An integrated, Gemini-powered AI assistant that provides real-time support, menu recommendations, and booking assistance.
*   **💳 Secure Checkout**: Fully integrated with the **Interswitch Payment Gateway**, offering a secure and optimized transaction flow with instant visual feedback.
*   **🍱 Meal Plan Management**: Structured subscription plans for weekly organic meal deliveries, tailored for various lifestyles.
*   **📅 Table Reservations**: A robust booking system that allows clients to secure their dining experience with ease.
*   **👤 Personal Dashboard**: A responsive customer account area for tracking order history, saved items, and profile management.
*   **💎 Premium UI/UX**: Built with vanilla CSS for maximum performance, featuring a luxury color palette, smooth animations, and a fully responsive design for all devices.

## 🛠️ Technical Stack
*   **Frontend**: Semantic HTML5, Vanilla JavaScript (ES6+), FontAwesome iconography.
*   **Backend**: Node.js, Custom HTTP Server.
*   **Database**: Supabase (PostgreSQL) for secure user authentication and order management.
*   **Payments**: Interswitch WebPay Integration.
*   **AI**: Google Gemini Pro 1.5.

## 🚀 Getting Started
1. **Clone & Install**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Nature-s-Cravings-App.git
   cd Nature-s-Cravings-App
   npm install
   ```
2. **Environment Setup**:
   Create a `.env` file with your `GEMINI_API_KEY`.
3. **Launch**:
   ```bash
   node server.js
   ```
